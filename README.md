## 📌 En qué consiste el proyecto actual

Sistema web fullstack para la gestión integral de bares. Incluye control de stock, comandas por mesa, menú digital y escaneo inteligente de boletas de proveedores usando IA. Está pensado para escalar a múltiples clientes con datos completamente aislados entre sí e instalarse como app nativa desde el navegador sin pasar por ninguna tienda.
